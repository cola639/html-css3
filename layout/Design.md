the **most common baseline** for desktop&laptop is:

- **1440 × 900** (very common “desktop web / laptop” baseline)
- **1440 × 1024** (also very common, especially for dashboard/admin layouts)

If you want one default that matches most modern desktop UI kits:

- ✅ **1440px width** is the most commonly used desktop design width.
- For height, pick **900** (more laptop-realistic) or **1024** (more roomy dashboard).

A practical set many teams use:

- **Design:** 1440 × 900
- **Check small:** 1366 × 768
- **Check large:** 1920 × 1080 (make sure it doesn’t look stretched; use a max-width container)

the most common “baseline” frames for **mobile UI in Figma**, :

### Most common baseline (today)

- **390 × 844** — iPhone 12/13/14 (very popular default for modern iOS designs)

### Also very common (still used a lot)

- **375 × 812** — iPhone X/11 Pro/12 mini style baseline (many older design systems)

### If you want a “covers most Android” baseline

- **360 × 800** (or **360 × 780**) — common Android _dp_ baseline used in Material-style layouts

### Good practical set

- **Design:** 390 × 844
- **Check small Android:** 360 × 800
- **Check large phone:** 414 × 896 (or 430 × 932 for newer large iPhones)
