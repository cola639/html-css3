A responsive article layout with a sticky table of contents

Common blog structure: header, main article, and a sidebar that becomes a stacked section on mobile. The TOC is sticky on desktop for easier reading.

8 min read • Updated Jan 24, 2026 • By Taylor

1) Introduction

This is a plain HTML + Tailwind example. On small screens, the sidebar content moves below the article. On larger screens, it sits to the right and can be sticky.

2) Structure

Use a grid with lg:grid-cols-12 and assign spans for content and aside. The table of contents uses lg:sticky.

Example callout

Keep line-length comfortable for reading. A max width plus padding improves readability across devices.

3) Common patterns

  • Sticky TOC / related posts on desktop
  • Author card and newsletter CTA
  • Footer links and category navigation

4) Conclusion

This layout is great for documentation, knowledge base articles, and long-form posts. You can reuse the same structure in React/Vue by splitting it into components.
